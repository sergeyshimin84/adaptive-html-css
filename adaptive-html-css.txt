1.1 Применяем БЭМ (Блок, Элемент, Модификатор) — компонентный подход к веб-разработке, 
в основе которого лежит принцип разделения интерфейса на независимые блоки.
Предварительно нужно внимательно изучить макет сайта (страрнцы).
Выстроит стратегию верстки. Произвести анализ макета. Изначально заложить адаптив.
Выставить направляющие линии в макете (для начала отобразить линейку).
Примерная условно принятая ширина в макетах 1440px.
При верстке больших проектов компоненты сайта верстаются отдельно, вплоть до кнопок (отдельных элементов).
1.2 В проекте сбрасываем стили до дефолтных.
В макете Figma рекомендуется установить плагин Font Fascia для работы со шрифтами.
Устанавливаем в VSC плагин для работы со шрифтами Google Fonts.
В подключенном шрифте прописываем амперсант &display=swap (влияет на оптимизацию шрифтов, ускоряет загрузку).
Не забуть указать шрифт в файле стилей CSS (font-family).
//Хорошей практикой является проверка/тестирование изменений по ходу разарботки.
